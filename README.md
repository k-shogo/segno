# Minhash

An implementation of the b bit MinHash algorithm in ruby

## Installation

Add this line to your application's Gemfile:

    gem 'b-bit-minhash'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install b-bit-minhash
